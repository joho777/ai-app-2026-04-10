# BranchStack Lab

BranchStack Lab is a single-file, interactive “social version control” simulator inspired by GitButler’s vision of what comes after Git: stacked branches, real-time awareness, context as a first-class object, and AI agents working alongside humans.

## Live demo

After GitHub Pages is enabled, open:

- https://joho777.github.io/ai-app-2026-04-10/

## What you can do

- Add humans and AI agents
- Create branches and stack them
- Generate changes (patches) that touch files and accumulate conflict risk
- Merge low-risk branches to trunk (merge blocks if risk is too high)
- Add social context items (notes, issues, chat)
- Run a tiny “read-before-code” agent to summarize, plan, and generate a context-aware patch

## Notes

- Everything runs locally in your browser. No external services.
- Single `index.html` with embedded CSS + JS.
